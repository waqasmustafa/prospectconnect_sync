üéØ Sync Capabilities

  • Bi-directional sync - Changes flow both ways automatically
  • Field mapping - Assignee, lead source, tags, country, pipeline/stage
  • User mapping - Map Odoo users to ProspectConnect users
  • Pipeline mapping - Map CRM stages to ProspectConnect pipelines
  • Conflict resolution - Timestamp-based conflict handling
  • Duplicate prevention - Uses ProspectConnect IDs to prevent duplicates
  • Incremental sync - Every 5 minutes (configurable)
  • Nightly reconciliation - Daily at 2 AM for data integrity
  • Error handling - Automatic retry with detailed error logging
  • Job monitoring - Track sync status and troubleshoot issues
